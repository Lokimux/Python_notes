Your guide to Python from Zero to Hero!

1. What is Python?

Python is a versatile, easy-to-learn programming language used for various applications.

print("Hello, World!")

2. Variables

Variables are containers for storing data. They make it easy to reuse and manipulate values.


x = "Hello, World!"  # Storing "Hello, World!" in a variable
print(x)  # Output: Hello, World!

3. Input and Output

Take user input and display it using Python's input() and print() functions.


user_input = input("Enter something: ")  # Input a message
print("You entered:", user_input)  # Output the user's input

4. Decision Making (if, elif, else)

Control the flow of your program based on conditions.


message = "Hello, World!"

if len(message) > 10:
    print("Message is long:", message)
else:
    print("Message is short.")

5. Loops

Use loops to repeat actions.

For Loop


for i in range(3):
    print("Hello, World!")  # Prints three times

While Loop


count = 0
while count < 3:
    print("Hello, World!")  # Prints three times
    count += 1

6. Lists

A list is an ordered collection of items.


greetings = ["Hello, World!", "Hi, World!", "Hey, World!"]

for greeting in greetings:
    print(greeting)  # Prints each greeting

7. Tuples

Tuples are immutable lists.


greetings = ("Hello, World!", "Hi, World!", "Hey, World!")
print(greetings[0])  # Prints: Hello, World!

8. Dictionaries

Dictionaries store key-value pairs.


greeting_dict = {"English": "Hello, World!", "Spanish": "Hola, Mundo!"}
print(greeting_dict["English"])  # Prints: Hello, World!

9. Functions

Functions are reusable blocks of code.


def greet():
    return "Hello, World!"

print(greet())  # Calls the function and prints: Hello, World!

10. Classes

Classes define blueprints for objects.


class Greeting:
    def __init__(self, message):
        self.message = message

    def display(self):
        print(self.message)

g = Greeting("Hello, World!")
g.display()  # Prints: Hello, World!

11. Strings

Strings are sequences of characters enclosed in single or double quotes.


        # String Examples
        greeting = "Hello, World!"
        print(greeting.upper())  # Converts to uppercase: HELLO, WORLD!
        print(greeting.lower())  # Converts to lowercase: hello, world!
        print(greeting[0:5])     # Slicing: Hello

12. Error Handling

Use try-except blocks to handle errors gracefully.


        try:
            x = int("Hello")
        except ValueError:
            print("Invalid conversion!")  # Prints: Invalid conversion!
        finally:
            print("Execution finished.")

13. File Handling

Python provides functions to read, write, and manipulate files.


        # Writing to a file
        with open("example.txt", "w") as file:
            file.write("Hello, World!")
        
        # Reading from a file
        with open("example.txt", "r") as file:
            content = file.read()
            print(content)  # Prints: Hello, World!

14. List Comprehension

List comprehensions are a concise way to create lists.


        # Create a list of squares
        squares = [x**2 for x in range(5)]
        print(squares)  # Prints: [0, 1, 4, 9, 16]

15. Lambda Functions

Lambda functions are small, anonymous functions.


        # Lambda function to add two numbers
        add = lambda x, y: x + y
        print(add(2, 3))  # Prints: 5

16. Modules

Modules are files containing Python code that can be reused.


        # Importing a module
        import math
        print(math.sqrt(16))  # Prints: 4.0

17. Decorators

Decorators are functions that modify the behavior of other functions.


        def decorator(func):
            def wrapper():
                print("Before the function")
                func()
                print("After the function")
            return wrapper
        
        @decorator
        def say_hello():
            print("Hello, World!")
        
        say_hello()

18. Generators

Generators produce values one at a time using the `yield` keyword.


        def count_up_to(maximum):
            count = 1
            while count <= maximum:
                yield count
                count += 1
        
        for num in count_up_to(5):
            print(num)  # Prints: 1, 2, 3, 4, 5

19. Advanced Topics

1. Python Virtual Environment

Create isolated environments for Python projects.


        # Create a virtual environment
        python -m venv myenv
        
        # Activate the virtual environment (Linux/Mac)
        source myenv/bin/activate
        
        # Activate the virtual environment (Windows)
        myenv\Scripts\activate

2. Asyncio

Asyncio is used for asynchronous programming.


        import asyncio
        
        async def say_hello():
            await asyncio.sleep(1)
            print("Hello, World!")
        
        asyncio.run(say_hello())  # Prints: Hello, World! after 1 second
